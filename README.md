# Palo Alto, CAThis is a repo for managing machine readable artifacts for the [Palo Alto, CA APIs](http://paloalto.opendata.junar.com/developers/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Palo Alto, CA APIs](http://paloalto.opendata.junar.com/developers/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://paloalto.opendata.junar.com/developers/)- [Developer](http://paloalto.opendata.junar.com/developers/)- [Twitter](https://twitter.com/junar)The [APIs.json](https://github.com/api-evangelist/palo-alto-ca/blob/master/apis.json) index provides links to all the relevant links for the Palo Alto, CA APIs, including the OpenAPI and Postman if they are present.
